isEmailAddress

  • isEmailAddress(def?: EmailAddressDefinition): StringGuard
    • Defined in guards/string-guard.ts:225

    Checks if string is an email address number.

    remarks

    Chainable method.

    def:
- quick : Common implementation matching 99% of all email addresses in actual use today.
- rfc5322 : Lightened RFC 5322 implementation matching 99.99%.

Rules:
Lightened RFC 5322 (sections 3.2.3 and 3.4.1) and RFC 5321 implementation is omitting :
- IP addresses.
- domain-specific addresses.
- the syntax using double quotes and square brackets.
    see

    https://en.wikipedia.org/wiki/Email_address#Syntax for syntax.

    see

    http://www.regular-expressions.info/email.html for regex details.

    example

    John.Doe@example.com

    Parameters

    • Optional def: EmailAddressDefinition

      'quick' | 'rfc5322'. Default is 'quick'.

    Returns StringGuard
